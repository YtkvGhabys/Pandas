# 📊 Análise de Dados com Pandas

## Sobre o Projeto

Este projeto reúne diferentes análises exploratórias e processos de tratamento de dados utilizando a biblioteca **Pandas** em **Python**. O objetivo foi aplicar conceitos fundamentais de manipulação de dados, desde o carregamento de datasets até a geração de insights por meio de filtragens, transformações, cálculos estatísticos e visualizações.

---

## Habilidades Demonstradas

- Importação e exploração de datasets
- Limpeza e tratamento de dados
- Tratamento de valores nulos
- Filtragem de informações
- Criação de novas colunas e métricas
- Estatísticas descritivas
- Exportação de dados tratados
- Visualização de resultados

---

# Análise do Dataset de Alunos

## Limpeza e Preparação dos Dados

Durante o processo de preparação dos dados foram realizadas as seguintes etapas:

- Tratamento de valores nulos, substituindo-os por `0`;
- Remoção de registros específicos;
- Criação de um novo dataset contendo apenas os alunos aprovados;
- Exportação da base de dados tratada.

## Principais Insights

A análise mostrou que a aplicação de **pontos extras** pode alterar significativamente o status de aprovação de alguns estudantes, evidenciando como critérios adicionais de avaliação podem impactar diretamente os resultados finais.

# Análise do Dataset de Imóveis

## Limpeza e Transformação

Para tornar a análise mais consistente, foram realizadas as seguintes etapas:

- Remoção de imóveis comerciais;
- Filtragem apenas de apartamentos;
- Organização dos dados para análises estatísticas.

## Estatísticas Descritivas

Foram calculadas diversas métricas importantes, entre elas:

- Média de quartos por apartamento;
- Quantidade de bairros distintos presentes no dataset;
- Estatísticas gerais sobre os imóveis.

## Principais Insights

A análise permitiu identificar os bairros com os **maiores valores médios de aluguel**, fornecendo informações úteis para:

- Investidores;
- Estudos sobre valorização imobiliária;
- Comparação entre regiões;
- Análises do mercado de locação.

## Resultados

Ao final do projeto foi possível:
